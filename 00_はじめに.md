# スピリチュアル系リサーチ — はじめに

エージェント向けの編集ルール・ID・必須更新は [[AGENTS]] を参照。

## このフォルダのゴール

**ジャンルシート**（`20_ジャンル/G###/…/_genre_sheet.md`）に、市場の全体像・バズの型・注目チャンネルを集約する。  
ここが **結論の置き場**（SoT）。チャンネル調査や動画1本のメモは **素材** であり、最終的にジャンルシートへ反映する。

---

## ベストな使い方（推奨フロー）

1. **対象ジャンルを決める** … `G###` を `_genre_registry.md` と `[[20_ジャンル/_genre_index]]` で確認（なければ採番して追加）。
2. **分析したいチャンネルを指定する** … URL または `CH###`（未登録なら台帳に行を足し、`30_チャンネル調査/` にフォルダを作る）。
3. **AIに依頼する** … 「`10_テンプレート/チャンネル分析_ジャンルへ転記用_テンプレート.md` を埋め、続けて `G###` の `_genre_sheet.md` の §4〜6 と §5 の表を更新して」と渡す。  
   - 動画の細部まで見る場合は、あわせて `動画1本リサーチ_テンプレート` 用に `VID###/research.md` を用意すると精度が上がる。
4. **比較・コンセプト** … 勝ち負けを見るときは `40_比較とコンセプト/`。自チャンネル案は `コンセプト反映.md`。

※ 手順の要約だけ見る場合は、下の「フォルダ早見表」を使う。
